# spring-boot-demo

## 项目介绍

基于 spring boot 2.X 开发，主要用于造轮子。

### 目录结构
```
spring-boot-demo
|-- common 公共模块
|-- ucenter 用户系统
|-- kapi 接口管理系统
|-- s-registry 注册中心
```

## 涉及技术纲目

- 数据库`mysql`、`mongoDB`
- [MyBatis-Plus](http://mp.baomidou.com)
    > 是一个 MyBatis 的增强工具，在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。
